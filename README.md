# airflow-dlt-s3-data-pipeline
Este projeto implementa um pipeline de dados completo, simulando um cenário real de engenharia de dados.  A pipeline realiza a ingestão de dados de uma API de câmbio (FreeCurrency API), processa os dados e armazena em um data lake no formato Parquet utilizando MinIO (S3 local), com orquestração via Apache Airflow.
